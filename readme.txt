The zip contains the full app complete with all needed dependencies that can be found in 
/__pycache__ and /flask respective directories.

The app is named mp_stories.py, followed by an html page to be found in /templates and a css file in /static.
To run the app follow to following steps:
Assuming you have python environment set up on your system, use terminal to direct yourself to the current folder and type:

1- "source flask/bin/activate" to load the environment for the app

2- now you can run the app, so type "python mp_stories.py"

Open your browser(reccommended firefox) and go to the following URL: "http://http://127.0.0.1:5000/" to access the web app. Enjoy
